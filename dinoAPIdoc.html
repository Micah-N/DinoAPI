Available commands and how to use

GET

Used to search based on specified parameter. Search a single entity, all entities or parameter qualified entries. Can be used in both Admin and User modes.

How to test with Postman (In Admin Mode)

  • Enter url: micahnetz.com/dinoAPI/commandsAdmin.php
  • Select request method: GET
  • To view all entries leave all params empty.
  • To search by input parameter, enter the following params:
    • keyword: the parameter you want to search (i.e. Name)
    • name: the name you want to use for the search
    • order: the order you want to use for the search
    • suborder: the suborder you want to use for the search
    • when: the era you want to use for the search
    • where: the location you want to use for the search
    • food: the diet you want to use for the search

How to structure query (Admin)

$.ajax({
	type: "GET",
	dataType: "json",
	async: false,
	url: "http://micahnetz.com/dinoAPI/commandsAdmin.php",
	data:
	{
		'name': name,
		'order': order,
		'suborder': suborder,
		'when': when,
		'where': where,
		'food': food,
		'keyword': keyword
	},
	complete: function(data){
		/*report success (returns json results of GET request)*/;
	}
});

PUT

Used to update/edit a single data entry. Can only be used in Admin mode. Use the id variable to specify the id # of the dinosaur entry to update.

How to test with Postman (In Admin Mode)

  • Enter url: micahnetz.com/dinoAPI/commandsAdmin.php
  • Select request method: PUT
  • Enter the following params:
    • id: the id # of entry to be edited
    • name: the name you want to replace original name with
    • order: the order you want to replace original order with
    • suborder: the suborder you want to replace original suborder with
    • when: the era you want to replace original era with
    • where: the location you want to replace original location with
    • food: the diet you want to replace original diet with

How to structure query (Admin)

$.ajax({
	type: "PUT",
	dataType: "json",
	url: "http://micahnetz.com/dinoAPI/commandsAdmin.php?id=" + id
	+"&name=" + name
	+ "&order=" + order
	+ "&suborder=" + suborder
	+ "&when=" + when
	+ "&where=" + where
	+ "&food=" + food,
	complete: function(data){
		/*report success*/
	}
});
