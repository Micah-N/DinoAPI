Available commands and how to use

GET
GET: Used to search based on specified parameter. Search a single entity, all entities or parameter qualified entries. Can be used in both Admin and User modes.
How to test with Postman (In Admin Mode)
Enter url: micahnetz.com/dinoAPI/commandsAdmin.php
Select request method: GET
To view all entries leave all params empty
To search by input parameter (be sure to specify the keyword - what type parameter to search)
Enter the following params:
key: keyword
value: the parameter you want to search (i.e. Name)
key: name
value: the name you want to use for the search
key: order
value: the order you want to use for the search
key: suborder
value: the suborder you want to use for the search
key: when
value: the era you want to use for the search
key: where
value: the location you want to use for the search
key: food
value: the diet you want to use for the search
How to structure query (Admin): $.ajax({ type: "GET", dataType: "json", async: false, url: "http://micahnetz.com/dinoAPI/commandsAdmin.php", data: { 'name': name, 'order': order, 'suborder': suborder, 'when': when, 'where': where, 'food': food, 'keyword': keyword }, complete: function(data){ /*report success (returns json results of GET request)*/; } });
How to structure query (User): $.ajax({ type: "GET", dataType: "json", async: false, url: "http://micahnetz.com/dinoAPI/commandsUser.php", data: { 'name': name, 'order': order, 'suborder': suborder, 'when': when, 'where': where, 'food': food, 'keyword': keyword }, complete: function(data){ /*report success (returns json results of GET request)*/ } });
POST
POST: Used to add a data entry. Can only be used in Admin mode.
How to test with Postman (In Admin Mode)
Enter url: micahnetz.com/dinoAPI/commandsAdmin.php
Select request method: POST
Enter the following params:
key: name
value: the name you want to use for the new entry
key: order
value: the order you want to use for the new entry
key: suborder
value: the suborder you want to use for the new entry
key: when
value: the era you want to use for the new entry
key: where
value: the location you want to use for the new entry
key: food
value: the diet you want to use for the new entry
How to structure query (Admin): $.ajax({ url: "http://micahnetz.com/dinoAPI/commandsAdmin.php", data: { "name": name, "order": order, "suborder": suborder, "when": when, "where": where, "food": food }, type: "POST", complete: function(){/*report success*/}, });
PUT
PUT: Used to update/edit a single data entry. Can only be used in Admin mode. Use the id variable to specify the id # of the dinosaur entry to update.
How to test with Postman (In Admin Mode)
Enter url: micahnetz.com/dinoAPI/commandsAdmin.php
Select request method: PUT
Enter the following params:
key: id
value: the id # of entry to be edited
key: name
value: the name you want to replace original name with
key: order
value: the order you want to replace original order with
key: suborder
value: the suborder you want to replace original suborder with
key: when
value: the era you want to replace original era with
key: where
value: the location you want to replace original location with
key: food
value: the diet you want to replace original diet with
How to structure query (Admin): $.ajax({ type: "PUT", dataType: "json", url: "http://micahnetz.com/dinoAPI/commandsAdmin.php?id=" + id +"&name=" + name + "&order=" + order + "&suborder=" + suborder + "&when=" + when + "&where=" + where + "&food=" + food, complete: function(data){ /*report success*/ } });
DELETE
DELETE: Used to delete a single data entry. Can only be used in Admin mode. Use the id variable to specify the id # of the dinosaur entry to delete.
How to test with Postman (In Admin Mode)
Enter url: micahnetz.com/dinoAPI/commandsAdmin.php
Select request method: DELETE
Enter the following params:
key: id
value: the id # of entry to be deleted
How to structure query (Admin): $.ajax({ url: "http://micahnetz.com/dinoAPI/commandsAdmin.php?id=" + id, type: "DELETE", dataType: "json", complete: function(data){ /*report success*/ } });
